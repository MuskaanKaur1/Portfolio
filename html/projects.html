1.Ecommerce Website

Ecommerce website built using the MERN stack (MongoDB, Express, React, Node.js). It includes user authentication, product browsing, wishlist functionality and cart functionality.Also contain admin panel that can upload category and product details which appear in the frontend.

Key Features

  • User Signup/Login with JWT Authentication

  • Admin panel to manage categories and products by add,delete and update (with image upload)
  • Admin panel has Responsive UI for both desktop and mobile

3.Chatbot

This chatbot is a React-based AI assistant that interacts with users using Google's Gemini API to generate real-time responses. It features a popup chat interface, tracks conversation history, and provides smooth user interactions. The chatbot can process user messages, send them to the API, and display AI-generated responses.


Key Features

  • AI-Powered Responses – Uses Gemini API for smart replies.
  • Real-time Chat – Sends and receives messages dynamically.
  • Auto-scroll – Chat window scrolls to the latest message.
  • User-Friendly UI – Clean design with a chatbot icon.
  • State Management – Uses useState and useEffect for smooth updates.

Technologies & Languages Used: React.js – For building the UI. JavaScript (ES6+) – Manages app logic and API requests. CSS – Styles the chatbot interface. Vite – A fast build tool for development. Google Gemini API – Generates chatbot responses.

4.Smoke Sensor

A smoke detector is a device that senses smoke, which is often an early indicator of fire. In this IoT project, we use the MQ2 smoke sensor with an Arduino Uno to create a simple yet powerful fire warning system.

The MQ2 sensor can detect harmful gases like LPG, methane, propane, alcohol, hydrogen, carbon monoxide, and general smoke. It sends this information to the Arduino.

When smoke is detected:
  • The analog signal increases.
  • The Arduino reads this signal.
  • If it crosses a safe limit, the buzzer turns ON, making a loud noise to alert people nearby.

This system helps save lives and property by giving an early warning in case of fire or gas leaks. It’s a basic but effective IoT safety solution that can be used in homes, kitchens, offices, and factories.
